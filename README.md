# 🤖 AI Chatbot Frontend

A responsive AI chatbot user interface built with ReactJS. The application provides a clean chat experience and communicates with a backend API to generate AI-powered responses.

## Features

- Modern chat interface
- Real-time message display
- Responsive design for desktop and mobile
- Loading indicators for AI responses
- Easy integration with backend APIs

## Tech Stack

| Technology | Purpose |
|------------|----------|
| ReactJS | Frontend Framework |
| JavaScript | Application Logic |
| HTML5 | Structure |
| CSS3 | Styling |
